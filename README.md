# KIRA-PROYECTO-DE-GRADO
KIRA es un sistema híbrido de retroalimentación que combina una aplicación web y una caja física en El Espinal, Tolima. Facilita a los clientes registrar su satisfacción y permite a los administradores acceder a reportes en tiempo real para mejorar la calidad del servicio y fomentar la mejora continua.
